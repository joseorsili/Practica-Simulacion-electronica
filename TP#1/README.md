#### Trabajo Practico #1

##### Hardware y Software libre

Verificar los siguientes conceptos:
1. ¿Qué es el software libre?
    El software libre se refiere a programas de computadora que se distribuyen con una licencia que permite a los usuarios la libertad de usar, copiar, distribuir y modificar el software sin restricciones. Estas libertades están respaldadas por cuatro libertades esenciales:
    La libertad de usar el software para cualquier propósito.
    La libertad de estudiar cómo funciona el software y modificarlo según sea necesario.
    La libertad de distribuir copias del software a otros usuarios.
    La libertad de mejorar el software y publicar las mejoras para que otros las utilicen.
    El software libre es una alternativa al software propietario, que es controlado por empresas privadas y suele venir con limitaciones en cuanto a su uso y modificación. El software libre se basa en una comunidad de colaboración y puede ser modificado y mejorado por cualquier persona, lo que lo convierte en una opción popular para muchos desarrolladores y usuarios de todo el mundo. Ejemplos de software libre incluyen Linux, Apache, Firefox y LibreOffice.
2. ¿Qué es el Hardware libre?
    El hardware libre es similar al software libre, pero se refiere a diseños de hardware, como placas de circuito, componentes electrónicos, dispositivos y sistemas, que se distribuyen con una licencia que permite a los usuarios la libertad de utilizar, estudiar, modificar y distribuir el diseño del hardware sin restricciones.
    Al igual que el software libre, el hardware libre se basa en la colaboración y la transparencia, y se centra en la creación de un ecosistema de conocimiento y recursos compartidos para mejorar el hardware existente y desarrollar nuevos diseños. Los diseños de hardware libre suelen estar disponibles públicamente en línea, y la comunidad de usuarios puede colaborar en la mejora de los diseños, crear variantes personalizadas o producir productos comerciales basados en ellos.
    El movimiento del hardware libre se ha vuelto más popular en los últimos años, y algunos ejemplos de hardware libre incluyen la plataforma de prototipado Arduino, la placa de circuito impreso BeagleBone Black y la cámara digital de código abierto Apertus AXIOM.
3. ¿Cuál es la diferencia entre hardware y software libre?
    La principal diferencia entre hardware y software libre radica en su naturaleza. El software libre es un conjunto de programas y código que se pueden copiar, distribuir y modificar sin restricciones, mientras que el hardware libre se refiere a diseños de hardware que se pueden utilizar, estudiar, modificar y distribuir sin restricciones.
    Además, el software libre se distribuye en forma de código fuente, que puede ser modificado por cualquier persona con conocimientos de programación, mientras que el hardware libre se distribuye en forma de diseño, que puede ser utilizado para fabricar componentes o sistemas físicos.
    Otra diferencia importante es que el software libre se ejecuta en hardware existente, mientras que el hardware libre es el propio hardware. Esto significa que los usuarios pueden descargar, instalar y utilizar software libre en su computadora, teléfono inteligente u otro dispositivo existente, pero necesitan un conjunto de componentes físicos para utilizar el hardware libre.
    En resumen, el software libre se refiere a programas y código que se pueden copiar, distribuir y modificar sin restricciones, mientras que el hardware libre se refiere a diseños de hardware que se pueden utilizar, estudiar, modificar y distribuir sin restricciones. Ambos se basan en la colaboración y la transparencia, y se centran en la creación de un ecosistema de conocimiento y recursos compartidos para mejorar el software y el hardware existentes y desarrollar nuevos diseños.
4. ¿Qué proyectos de Hardware libre conoce?
    Hay muchos proyectos interesantes de hardware libre en diferentes áreas, aquí mencionaré algunos de los más conocidos y populares:
    1. Arduino: Es una plataforma de prototipado electrónico que se compone de placas y software de código abierto. Se ha convertido en una herramienta popular para proyectos de electrónica, robótica y domótica.
    2. Raspberry Pi: Es un pequeño ordenador de placa única que se utiliza para una variedad de aplicaciones, desde la educación y la programación hasta la automatización del hogar y la creación de servidores.
    3. BeagleBone: Es otra placa de desarrollo de código abierto que se utiliza para una variedad de proyectos de electrónica y robótica. Tiene un procesador más potente que la Raspberry Pi y se utiliza para aplicaciones de IoT y automatización.
    4. OSHWA: Es una organización que promueve el hardware abierto y mantiene una base de datos de hardware de código abierto.
    5. RepRap: Es un proyecto de impresión 3D de código abierto que permite a los usuarios construir y mejorar sus propias impresoras 3D. 
    6. KiCad: Es una suite de software libre y de código abierto para diseño de circuitos electrónicos, que incluye herramientas para diseño esquemático, diseño de placas de circuito impreso y visualización 3D.
    7. Libreboot: Es un proyecto que busca proporcionar firmware libre y de código abierto para sistemas basados en la arquitectura x86, eliminando el software propietario en los BIOS y UEFI.
    8. Apertus AXIOM: Es una cámara digital de cine de código abierto, diseñada para ser modular y adaptable a diferentes necesidades de producción.
    Estos son solo algunos ejemplos de proyectos de hardware libre, hay muchos más proyectos interesantes y en constante desarrollo en diferentes áreas.
5. ¿Qué es Arduino?
    Arduino es una plataforma de hardware libre y software libre de código abierto diseñada para facilitar la creación de proyectos electrónicos interactivos. Consiste en una serie de placas electrónicas y un entorno de programación que permite a los usuarios crear y programar dispositivos electrónicos interactivos y robots de manera fácil y accesible.
    Arduino se basa en una placa de circuito impreso que contiene un microcontrolador y una serie de entradas y salidas, incluyendo puertos analógicos y digitales, interfaces serie y USB. Los usuarios pueden programar la placa usando el entorno de desarrollo integrado (IDE) de Arduino, que está disponible de forma gratuita en línea y es compatible con Windows, Mac OS X y Linux.
    El IDE de Arduino utiliza un lenguaje de programación simplificado basado en C++ y se centra en la facilidad de uso y la accesibilidad para usuarios sin experiencia en programación. Los usuarios pueden cargar programas en la placa Arduino a través de un cable USB, lo que les permite crear dispositivos que interactúan con el mundo físico de diversas formas, como leer sensores, controlar motores y luces, y enviar datos a través de la red.
    Arduino se ha convertido en una herramienta popular para la educación en electrónica y programación, así como para proyectos de robótica, automatización del hogar, arte interactivo y muchas otras aplicaciones creativas.
6. ¿ Cuál es el lenguaje de programación de Arduino? ¿Cuál es su IDE?
    El lenguaje de programación de Arduino está basado en C++ y se ha simplificado para hacerlo más fácil de usar para personas sin experiencia previa en programación. Arduino utiliza una serie de funciones predefinidas y una estructura de programación simplificada que permite a los usuarios enfocarse en la lógica de su proyecto sin preocuparse por la complejidad del lenguaje de programación subyacente.
    El entorno de desarrollo integrado (IDE) de Arduino es una aplicación que se utiliza para escribir, compilar y cargar programas en la placa de Arduino. El IDE es gratuito y está disponible para Windows, Mac OS X y Linux. Proporciona una interfaz de usuario gráfica intuitiva que incluye una ventana de edición de código, un área de mensajes de salida, un monitor serial y herramientas de carga de programas.
    El IDE de Arduino está diseñado para ser fácil de usar y se integra con la placa de Arduino de manera transparente, lo que permite a los usuarios cargar programas en la placa con un solo clic. También incluye una gran biblioteca de funciones y ejemplos de código que los usuarios pueden utilizar para empezar a crear proyectos rápidamente.
7. ¿Qué es Blue Pill? ¿Se puede programar con el ide de Arduino?
    Blue Pill es el nombre común de una placa de desarrollo de código abierto basada en el microcontrolador STM32F103C8T6, que es producida por varias empresas chinas. Es una placa pequeña y económica que cuenta con un microcontrolador ARM Cortex-M3 de 32 bits y 64 kB de memoria flash, lo que la hace adecuada para proyectos de electrónica de nivel intermedio y avanzado.
    Aunque el Blue Pill no está diseñado específicamente para funcionar con el entorno de desarrollo integrado (IDE) de Arduino, es posible programar la placa utilizando el IDE de Arduino. Para hacerlo, los usuarios necesitan instalar una biblioteca de compatibilidad de hardware y configurar el IDE para utilizarla.
    La biblioteca de compatibilidad de hardware se llama STM32duino y permite a los usuarios programar placas basadas en el microcontrolador STM32 utilizando el IDE de Arduino. La biblioteca incluye una gran cantidad de funciones y ejemplos de código para hacer que la programación del Blue Pill sea más fácil.
8. ¿Qué es Wemos D1 y Wemos ESP 32? ¿Se pueden programar con el ide Arduino?
    Wemos D1 y Wemos ESP32 son dos placas de desarrollo basadas en los microcontroladores ESP8266 y ESP32, respectivamente. Ambas placas son compatibles con el entorno de desarrollo integrado (IDE) de Arduino y son populares entre los aficionados y los desarrolladores de proyectos de IoT (Internet de las cosas) debido a su tamaño compacto, su bajo costo y su conectividad Wi-Fi.
    Wemos D1 está basado en el microcontrolador ESP8266, que cuenta con un procesador de 32 bits y Wi-Fi integrado. La placa D1 cuenta con 11 pines de entrada y salida digitales y un solo puerto analógico. También tiene una memoria flash de 4 MB y una interfaz USB.
    Wemos ESP32, por otro lado, está basado en el microcontrolador ESP32, que es más potente que el ESP8266 y también cuenta con conectividad Wi-Fi integrada. La placa ESP32 cuenta con 36 pines de entrada y salida digitales, 18 de los cuales pueden ser utilizados como pines de entrada analógicos. También tiene dos núcleos de procesamiento de 32 bits y 520 KB de memoria SRAM.
    Ambas placas pueden ser programadas utilizando el IDE de Arduino mediante la instalación de las bibliotecas adecuadas. La placa Wemos D1 puede ser programada utilizando la biblioteca ESP8266 para el IDE de Arduino, mientras que la placa Wemos ESP32 puede ser programada utilizando la biblioteca ESP32. Estas bibliotecas proporcionan una amplia gama de funciones y ejemplos de código que facilitan la programación de proyectos para estas placas.
9. ¿ Se puede programar Arduino con otros ide? ¿qué es platformio?
    Sí, Arduino se puede programar con otros IDE (Entornos de Desarrollo Integrado). De hecho, existen varias opciones de IDE que se pueden utilizar para programar placas Arduino, como Atmel Studio, CodeBlocks, Eclipse y Visual Studio, entre otros. Sin embargo, el IDE de Arduino es uno de los IDE más populares y ampliamente utilizados para programar placas Arduino.
    Por otro lado, PlatformIO es un ecosistema de desarrollo para la programación de microcontroladores, que soporta varias plataformas de hardware, incluyendo Arduino. PlatformIO proporciona un IDE multiplataforma basado en el editor de código Visual Studio Code y permite la programación de placas Arduino y otras plataformas mediante la instalación de las bibliotecas necesarias. Además, PlatformIO cuenta con un administrador de bibliotecas integrado, lo que facilita la búsqueda, instalación y gestión de bibliotecas para el desarrollo de proyectos.
10. ¿ Qué beneficios tiene programar con ide vscode?
    Visual Studio Code es un IDE de código abierto y gratuito que tiene varios beneficios para la programación de proyectos de hardware y software, incluyendo los proyectos basados en Arduino. Algunos de estos beneficios son:
    1. Extensibilidad: Visual Studio Code cuenta con una gran cantidad de extensiones y complementos disponibles que pueden ampliar las funcionalidades del IDE, lo que permite personalizarlo y adaptarlo a las necesidades específicas de cada usuario.
    2. Integración con Git: Visual Studio Code tiene una integración nativa con el sistema de control de versiones Git, lo que facilita el control y la gestión de las versiones del código fuente del proyecto.
    3. Depuración: Visual Studio Code cuenta con herramientas de depuración integradas que permiten a los usuarios depurar el código y encontrar errores con facilidad.
    4. Multiplataforma: Visual Studio Code es un IDE multiplataforma que está disponible para Windows, Mac y Linux, lo que permite a los usuarios utilizar el IDE en la plataforma de su elección.
    5. Inteligencia artificial: Visual Studio Code cuenta con características de inteligencia artificial que permiten a los usuarios autocompletar el código, buscar definiciones y referencias, refactorizar el código y más.
